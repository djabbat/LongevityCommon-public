# HAP — HepatoAffective Primacy Theory

**Теория гепатогенного происхождения эмоций**

**Author:** Dr. Jaba Tkemaladze | **Status:** Accepted for publication (*Biological Reviews*, IF ~10)

## Core Claim

Affective states cannot **develop** in any bilateral animal lacking an organ that:
1. Secretes steroid regulatory molecules crossing the BBB
2. Performs barrier-detoxification of circulating fluid

## Key Evidence

- Convergent evolution: bile acids (vertebrates) ↔ ecdysteroids (arthropods)
- FXR/TGR5 receptors in limbic structures
- Clinical correlates: hepatic encephalopathy, mood disorders in liver disease
- BHCA score: ~20/27 (Class II — strong circumstantial causation)
