## 1. Обзор проекта

Ze Vectors Theory предлагает, что любая информационно-обрабатывающая система — физическая, биологическая или вычислительная — может быть описана как бинарный поток счетчика двух комплементарных типов событий:

| Символ | Значение | Направление |
|--------|----------|-------------|
| **T** (Tension) | Событие превышает предсказание / порог | увеличение |
| **S** (Stretch) | Событие ниже предсказания / порога | уменьшение |

Теория служит **интерпретативной рамкой для квантовой механики**, а не выводом из первых принципов. Она принимает стандартный формализм КМ (гильбертовы пространства, POVM, правило Борна) как аксиоматический и добавляет операциональные правила для Ze-наблюдателей.

### 2.2. Пять постулатов (кратко)

1. **Формальная первичность Ze** — пространство-время выводится из сети Ze, не предполагается
2. **Собственное время (τ_Z)** — дискретный счетчик, наследуемый от родительского Ze
3. **Цена предсказания** — τ_Z уменьшается при неправильных прогнозах (T-событиях)
4. **Борьба за существование** — две стратегии выживания: фильтрация (внутренняя) и нишестроительство (внешняя)
5. **Наблюдение = Коллапс** — наблюдение Ze есть онтологический коллапс (в отличие от копенгагенского эпистемического)

### 2.3. Математический формализм

```
Ze = (H, ρ_Z, {M_i}, τ_Z, θ_Z)
```

- **H** — гильбертово пространство (dim < ∞)
- **ρ_Z** — состояние убеждений (матрица плотности)
- **{M_i}** — POVM, Σ M_i†M_i = I_H
- **τ_Z ∈ ℕ** — счетчик собственного времени
- **θ_Z ≥ 0** — порог предсказания

**Динамика:**
- S-событие (правильный прогноз): τ_Z(t+1) = τ_Z(t)
- T-событие (неправильный): τ_Z(t+1) = τ_Z(t) − 1
- Смерть при τ_Z = 0

> R6 (биостатистик): "MCID, установленный на N=12, не является статистически обоснованным ни по каким стандартам (ICC, SEM, SRD, anchor-based методы). Его использование в заявке EIC/ERC/Wellcome будет немедленно отклонено рецензентами с формулировкой 'statistically illiterate'."

```
MCID_χZe = 0.05  — ТОЛЬКО РАБОЧАЯ ГИПОТЕЗА ДЛЯ ПЛАНИРОВАНИЯ ВЫБОРКИ
                   НЕ ИСПОЛЬЗОВАТЬ В ПУБЛИКАЦИЯХ И ЗАЯВКАХ
```

**Текущее состояние:** N=12, тест-ретест 48 ч; SD ≈ 0.03; MCID = 1.7 × SD — методологически неверно

**Что требуется для валидного MCID (план v7):**

| Метод | Требования | Целевой N | Срок |
|-------|-----------|-----------|------|
| **SEM-based** | ICC тест-ретест; SEM = SD × √(1−ICC); MCID = 1.65 × SEM | N≥50, 2 визита | Q4 2026 |
| **SRD (Smallest Real Difference)** | SRD = SEM × 1.96 × √2; надёжный порог измерения | N≥50 | Q4 2026 |
| **Anchor-based** | Корреляция Δχ_Ze с PROM (SF-36, PROMIS-Fatigue); MCID = Δχ_Ze при MID якоря | N≥100, клин. исход | Q2 2027 |
| **Distribution-based** | Cohen's d=0.5 × SD_population; требует попул. норм | N≥200 | Q3 2027 |

**Требуемая выборка (power analysis):**
```
Для обнаружения MCID=0.05 при SD=0.03, α=0.05, мощность 80%:
  d = MCID/SD = 0.05/0.03 ≈ 1.67 (large effect)
  N_min ≈ 12 (pair t-test) → слишком мало для ICC-based MCID
  
Для SEM-based MCID (target ICC≥0.75, SEM≤0.015):
  N_ICC ≥ 50 субъектов с 2 визитами (Koo & Mae 2016, PMID 27330520)
  
Для anchor-based MCID с умеренной корреляцией r=0.40:
  N_anchor ≥ 100 субъектов с PROM + χ_Ze (Revicki et al. 2008)
```

**Правило до валидации:** χ_Ze следует описывать как "исследовательский биомаркер в фазе разработки метода (analytical validation phase)" — без клинических порогов.

#### v*_active — Статистический план (v7 fix, 2026-04-10)

**⚠️ ОБНОВЛЕНО (BUG-v9-1/2, 2026-04-11): I²=90.3% — pooling некорректен.**

v*_active = 0.456 был рассчитан как медиана по объединённым N=196 (Cuban EEG N=88 + Dortmund HRV N=108). Cochran's Q = 20.613 (df=2, p<0.0001), I²=90.3% → **высокая гетерогенность → pooling некорректен**.

**Правильная стратегия (вместо pooled median):**

```
Per-dataset bootstrap план:
  1. Cuban EEG (N=88): bootstrap B=10000, BCa CI для v*_cuban
  2. Dortmund HRV (N=60): bootstrap B=10000, BCa CI для v*_dortmund
  3. MPI-LEMON (N=30): bootstrap B=10000 (если данные доступны)
  4. Сравнить: H₀: v*_cuban = v*_dortmund (2-sample permutation test)
  5. Только если I²<50% → допустимо pooled estimate
  6. При I²>75% (текущий случай): отдельные оценки per dataset
```

**Стандартная фраза для грантов:** *"We observe v*≈0.456 in the Cuban EEG cohort (N=88, 95% CI pending). Cross-cohort generalisability of this value is uncertain (I²=90.3% across three datasets). Determining whether v*_active is universal or population-specific is a primary objective of WP1."*

**Требуемая выборка для отчётности:**
- Минимум: N≥50 (хватает для грубой оценки)
- Достаточно для публикации: N≥100 с независимой репликацией
- Стандарт ERC/Wellcome: N≥200 + внешняя когорта (UK Biobank EEG)

**Статус:** Cuban EEG Dataset публично доступен (Valdes-Sosa et al., figshare). Анализ BootstraCIDaI запланирован как P1-E (см. TODO.md).

**R4-v4 критика ρ_Z:** Операциональное определение ρ_Z = diag(p₀, p₁) предполагает **нулевую квантовую когерентность** (off-diagonal = 0). Это делает формализм ℂ² математически эквивалентным **классической теории вероятностей**. Квантовый формализм в данном пределе — декоративный. Пути выхода:
- **Вариант A (сохранить классику):** Явно заявить, что Ze использует только классические вероятности; отказаться от ℂ²-нотации в пользу P(state=0|history)
- **Вариант B (ввести когерентность):** Обосновать ненулевые off-diagonal элементы через нейральные корреляции (γ-синхронизация, inter-electrode coherence). Это требует измерений EEG-когерентности, а не только мощности спектра.

**ПРИНЯТОЕ РЕШЕНИЕ — Вариант A (v6 fix, 2026-04-10):**

Ze-теория принимает **Вариант A** для биологических применений (χ_Ze, EEG контекст):
- Состояние наблюдателя обозначается **P_Z = (p₀, p₁)** (классический вектор вероятностей), а не матрицей плотности ρ_Z с ℂ²-нотацией
- Символ ρ_Z сохраняется **только** в физическом формализме Ze (постулаты 1–5, частицы, пространство-время), где квантовый формализм является естественным
- Разделение нотации: P_Z (биологические Ze-агенты) ↔ ρ_Z (квантово-физические Ze)

**Обоснование Ze-аппарата при классическом пределе:**
Квантовый формализм (POVM, гильбертово пространство) сохраняется в теоретическом ядре Ze потому, что:
1. **Унификация:** единый формализм описывает и квантовые частицы (ρ_Z недиагональна), и классических биологических агентов (P_Z диагональна) — это не ошибка, а граничный случай
2. **Обобщаемость:** при появлении EEG-когерентных данных (Вариант B) нотация расширяется без смены аппарата
3. **Соответствие публикациям:** опубликованные статьи (DOI 10.65649/a874t352) используют POVM-формализм — изменение нотации в производных работах потребует явной оговорки

**Правило для будущих публикаций о χ_Ze:** использовать обозначение P_Z, явно указывать: *"In the biological (EEG) limit, we work with the diagonal (classical) sector of the Ze formalism: P_Z = (p₀, p₁), which reduces the density matrix to a classical probability vector."*

#### 2.3.4. τ_Z Операционализация через EEG: анализ выживаемости Ze-цепей (v9 fix, 2026-04-10)

*R2 (вычислительный нейробиолог) v9: "τ_Z определён как счётчик — но как его измерить в реальном EEG? Нет прямого отображения τ_Z ↔ наблюдаемый сигнал. Без операционализации теория нефальсифицируема."*

**Предлагаемое операциональное определение τ_Z через EEG:**

Определяем **Ze-цепь** (Ze-chain) как непрерывную последовательность правильных прогнозов (S-событий) в скользящем окне Born-rule классификатора:

```
Ze-chain(t) = максимальная длина непрерывной S-последовательности
              заканчивающейся в момент t

               τ_Z(t) ~ Ze-chain(t)   (прокси)
```

**Алгоритм вычисления прокси τ_Z из EEG:**

```
1. EEG сигнал → band-pass 25–35 Гц → бинаризация по порогу (квантиль 70%)
   → двоичная последовательность x(t) = {0,1}

2. Скользящее окно W=2c → оценка p₁(t) = #(1 в окне) / W

3. Прогноз Born-rule: ŷ(t) = 1 если p₁(t) > 0.5 else 0

4. T-событие: e(t) = 1 если x(t) ≠ ŷ(t)

5. Ze-chain(t) = длина серии e(t)=0 до момента t (обнуляется при e(t)=1)

6. Прокси: τ̂_Z(t) = Ze-chain(t); нормировка: τ̂_Z ∈ [0, τ₀]
```

**Анализ выживаемости Ze-цепей:**

Применяем анализ выживаемости Каплана-Мейера к длинам Ze-chain как событиям "гибели цепи":

```
Событие: Ze-chain прерывается (T-событие = смерть цепи)
Время: длина цепи в тактах
Censoring: конец записи = цензурирование

Вопрос: медианная выживаемость Ze-цепи при χ_Ze=1.0 vs χ_Ze=0.5?
Гипотеза: S₅₀(t) при χ_Ze>0.8 > S₅₀(t) при χ_Ze<0.4
Тест: log-rank + Cox PH с χ_Ze как ковариатой
```

**Связь с χ_Ze:**
```
χ_Ze(t) = 1 − |v(t) − v*| / max(v*, 1−v*)

Интерпретация через τ_Z-proxy:
  χ_Ze → 1  ⟺  v(t) ≈ v*  ⟺  длинные Ze-цепи (медленная смерть)
  χ_Ze → 0  ⟺  v(t) ≫ v* или v(t) ≪ v*  ⟺  короткие Ze-цепи
```

**Статус:** Алгоритм формализован; анализ на Cuban EEG Dataset (N=88) запланирован как часть P1-E (TODO.md). Требует Python-скрипт `scripts/tau_z_survival.py` (не создан).

**Предостережение:** τ̂_Z — прокси, не прямое измерение τ_Z. Формальное соответствие прокси ↔ постулируемый счётчик является фальсифицируемой гипотезой (нулевая: Ze-цепи не длиннее при χ_Ze=1 чем при χ_Ze=0).

#### Математическое сравнение Ze vs FEP (v8 fix — R4: "prose is not enough")

*R4 (физик-теоретик): "Сравнение с FEP/IIT должно быть на уровне уравнений, не только прозы."*

**Free Energy Principle (Friston 2010):**
```
F = E_q[log q(s|o) − log p(o,s)]
  = D_KL[q(s|o) ‖ p(s|o)] − log p(o)   (variational free energy)

Минимизация F ≡ минимизация prediction errors + complexity cost
Агент обновляет q(s|o) → argmin_q F(q, o)
```

**Ze Theory (Tkemaladze 2026):**
```
τ_Z(t+1) = τ_Z(t) − 1_T(t)   где 1_T = 1 если наблюдение ≠ прогноз Born-rule

Оптимальная стратегия (Теорема 5.1):
  q*(o_i) = Tr[ρ_Z M_i†M_i]  ≡  минимизация E[1_T]
           = argmin_q D_KL[q ‖ p_Born]   (при Born-rule = истинное распределение)
```

**Формальное соответствие:**

| Ze | FEP | Отношение |
|----|-----|-----------|
| τ_Z — дискретный ресурс, расходуется при T-событиях | F — непрерывная свободная энергия, минимизируется | Аналоги, не тождественны: Ze дискретен и конечен |
| Оптимум: q* = Born-rule (минимум D_KL[q‖p_Born]) | Оптимум: q* = posterior (минимум D_KL[q‖p(s\|o)]) | **Совпадают** в стационарной байесовской среде |
| τ_Z → 0: смерть Ze-агента | F → ∞: агент "сошёл с ума" (divergent beliefs) | Качественно аналогичны (resource exhaustion) |
| Нет иерархии; однослойный POVM | Иерархические предсказания (multiple timescales) | **Отличие: Ze не предполагает иерархии** |
| v* = информационный оптимум | F = Viterbi-like оптимальная траектория | Точка оптимума определена по-разному |

**Ключевое теоретическое утверждение (фальсифицируемо):**
Ze минимизация τ_Z расхода ≡ FEP-минимизация F **тогда и только тогда, когда**:
1. Среда стационарная (p(o|s) не меняется)
2. Prior p_Born = истинное распределение
3. Один иерархический уровень (без multi-scale)

Нарушение любого условия → Ze и FEP предсказывают разное поведение → **тестируемое различие**.

**Статус:** Математическое соответствие показано при упрощающих допущениях. Полное сравнение при нестационарной среде — открытая задача (2027).

#### Теоретический путь к выводу v*_active из первых принципов (R5 fix)

*Замечание R5: v*_active подобран эмпирически, а не выведен. Ниже — намеченный теоретический путь:*

**Подход 1 — Оптимальное управление с ресурсным ограничением:**
Рассматриваем Ze-агента как агента, максимизирующего ожидаемое время выживания J = E[τ_Z(T)] при стратегии предсказания π(ρ_Z). Оптимальная стратегия π* минимизирует частоту T-событий, что приводит к использованию Born-rule (Теорема 5.1). Однако в немарковской среде с изменяющимся ρ_Z существует нетривиальная оптимальная рабочая точка v*, определяемая уравнением:

```
∂J/∂v |_{v=v*} = 0   subject to:   τ_Z dynamics,  E[∑T-events] minimized
```

Замкнутая форма v* зависит от статистики среды (стационарный vs. нестационарный процесс). Для стационарного бинарного процесса: v*_passive = 1−ln2 (точно). Для нестационарного (биологический мозг): v*_active требует численного решения.

**Подход 2 — Информационная геометрия:**
v* — точка, в которой производительность KL-divergence-минимизирующей стратегии совпадает с производительностью равномерной случайной стратегии. Решение: v*_active = argmin_{v} [H(v) + D_KL(v‖uniform)] при условии τ_Z-constraint.

**Эмпирическое значение v*=0.456:** получено как медиана распределения оптимальных v по 196 субъектам Cuban EEG dataset + HRV данным Dortmund Vital Study. 95% ДИ не опубликован — открытая задача.

**Фальсифицирующий эксперимент (R5 рекомендация):** Если Ze-теория верна, транскраниальная стимуляция, смещающая v от v* в сторону θ=0.5 (случайного), должна снизить χ_Ze предсказуемо. Предлагаемый дизайн: TMS crossover study, N≥30, измерение χ_Ze до/после стимуляции. Этот эксперимент включён в Q2 2027 план (раздел 9).

#### Ze ↔ CDATA: математический мост (R5 fix)

*Рецензент R5 указал, что интеграция Ze и CDATA "заявлена вербально, математически не существует".
Ниже — первый шаг к формализации:*

**Концептуальная связь:** В CDATA клетка является Ze-наблюдателем с τ_Z(n) = оставшийся «предсказывательный ресурс», который убывает по мере накопления повреждений D(n). Математически:

```
τ_Z(n) = τ_Z(0) × Φ(D(n))
```

где Φ(D) — монотонно убывающая функция повреждения, Φ(0)=1, Φ(D_crit)→0.

**R4 fix (v4.0) — три конкурирующих функциональных формы:**

| Форма | Уравнение | Свойства | Биологическое обоснование |
|-------|-----------|----------|--------------------------|
| **L1 — Линейная** | Φ(D) = max(0, 1 − D/D_crit) | Простая; резкое обнуление; **clamp на нуле** | Простейшая аппроксимация, нет механизма |
| **E1 — Экспоненциальная** | Φ(D) = exp(−λ·D), λ = ln(2)/D₅₀ | Гладкая; τ_Z>0 всегда; Φ(D_crit)=1/e | Аналогична экспоненциальному распаду mito_shield в CDATA |
| **S1 — Сигмоидальная** | Φ(D) = 1 / (1 + exp(k·(D−D_crit))) | Порог с переходной зоной; биологически реалистична | Критический переход при накоплении повреждений (SASP-подобный) |

**ВАЖНО (R4-v4 fix):** В линейной форме L1 добавлен обязательный **clamp**:
```
τ_Z(n) = τ_Z(0) × max(0.0,  1 − D(n)/D_crit)
```
Без clamp τ_Z уходит в отрицательные значения при D > D_crit, что математически некорректно (τ_Z ∈ ℕ).

**Текущая позиция:** Используем L1 как нулевую гипотезу (простейшая). E1 (экспоненциальная) — приоритетная альтернатива, т.к. согласуется с mito_shield в CDATA. Выбор между L1/E1/S1 будет сделан по данным (регрессия τ_Z(n) ~ D(n) в пилотной когорте).

**Связь с χ_Ze:**
```
v*(n) ≈ v*₀ + Δv × (1 − Φ(D(n)))   →   χ_Ze(n) снижается от оптимума с возрастом
```

**Квантитативное предсказание для ERC/EIC (ERC-M3 fix, 2026-04-11):**

Для регистрации в ERC заявке используется E1-форма как наиболее биологически мотивированная:

```
τ_Z(n) = τ₀ · exp(−k · D(n))
```

где:
- τ₀ = начальный Ze-ресурс (свободный параметр, ~200 для стволовых клеток)
- k = скорость убыли Ze-ресурса (свободный параметр, определяется из регрессии; гипотетически k ~ α·ν / r из CDATA)
- D(n) = 1 − exp(−r·n) — CDATA центриолярное повреждение

**Фальсифицируемые предсказания формулы:**
1. Если k>0: ↑D → ↓τ_Z → ↑v (больше T-событий) → ↓χ_Ze (направление знака)
2. При D→D_crit (насыщение): τ_Z → τ₀·exp(−k) > 0 (экспоненциальная никогда не обнуляется — биологически реалистично)
3. Линейная корреляция: ln(χ_Ze) ~ −k·D, тестируемая на когорте с одновременным CHIP VAF + EEG

**Стандартная фраза для ERC §1:** *"We propose that the Ze proper-time resource τ_Z and the CDATA centriolar damage score D(n) are linked via τ_Z(n) = τ₀ · exp(−k · D(n)), predicting a monotonic decrease in χ_Ze with increasing D(n). Based on CDATA parameters (α=0.0082/division, ν_HSC≈0.01/day, D₅₀≈100 divisions), we estimate k ~ α·ν/r ≈ 0.023/division, predicting a 50% decrease in τ_Z by ~30 HSC divisions — testable in our WP2 longitudinal cohort with simultaneous EEG (χ_Ze) and CHIP VAF (D-proxy) at N≥100."*

**Численный пример (для ERC §1 credibility):**
```
D(n=30) = 1 − exp(−0.023×30) = 1 − exp(−0.69) ≈ 0.50
τ_Z(30) = τ₀ × exp(−0.023×30) ≈ τ₀ × 0.50

→ 50% снижение τ_Z за 30 делений HSC
→ χ_Ze снижается с ~0.87 (молодые) до ~0.73 (пожилые) согласно Cuban EEG данным
→ Совместимо: если χ_Ze ∝ τ_Z/τ₀ = exp(−k·D(n)), то 0.73/0.87 ≈ 0.84 = exp(−k·D_old)
→ D_old ≈ 0.18 при k=0.023, что соответствует n≈8 делениям — реалистично для HSC
```

**Статус:** Гипотеза. Требует:
1. Измерения χ_Ze в одной когорте с CDATA-биомаркерами (CHIP VAF, epigenetic age)
2. Регрессии ln(χ_Ze) ~ D(n) для оценки k
3. Проверки знака связи: ↑D → ↓χ_Ze (направление эффекта)

**Целевой срок формализации:** Q3 2026 (параллельно с UK Biobank данными)
